# ログ出力要件 (ID 14)

## 1. ライブラリ
- **SLF4J + Logback**: Spring Boot標準のログファサードおよび実装を使用。

## 2. ログレベルと運用
- **INFO**: ログイン成功/失敗、献立の登録・更新・削除、目標設定の変更など、ビジネス上の重要なイベント。
- **ERROR**: システム例外、スタックトレース、外部API通信失敗など、即時確認が必要な事象。

## 3. 出力形式
- **JSONフォーマット**: 将来的なクラウド監視（AWS CloudWatch LogsやGCP Cloud Logging等）でのパースを考慮。

## 4. 運用に関する補足
- ログの保存期間やローテーションポリシーは、インフラ設計フェーズで詳細を決定する。
