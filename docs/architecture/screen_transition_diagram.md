# 画面遷移図

## 遷移フロー

```text
[S-01 ログイン] -- 認証成功 --> [S-02 カレンダー]
               -- 認証失敗 --> [S-01 ログイン] (エラーメッセージ表示)
                                   |
                                   +-- 日付選択 --> [S-03 献立詳細・編集]
                                   |
                                   +-- 設定アイコン --> [S-04 設定・目標画面]
```

※ タイムアウトや未認証アクセス時は、強制的に [S-01 ログイン] へ遷移する。
