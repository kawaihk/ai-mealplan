# UI/UX 仕様書

## 1. 主要ユースケース (ID 16)
- **UC-01**: ユーザーがカレンダーから特定の日付を選択し、その日の献立（朝・昼・夕・間食）を登録する。
- **UC-02**: 登録済みの献立を別の日付にコピーする（時短機能）。
- **UC-03**: 1週間のPFCバランスと目標値の乖離をグラフで確認する。

## 2. 画面一覧・遷移 (ID 17)
| ID | 画面名 | 説明 | 遷移先 |
|:---|:---|:---|:---|
| S-01 | ログイン画面 | 認証を行う | S-02 |
| S-02 | カレンダー画面 | メイン画面。週間献立の概要を表示 | S-03, S-04 |
| S-03 | 献立詳細・編集 | 具体的なメニューと栄養素を入力 | S-02 |
| S-04 | 設定・目標画面 | ユーザー情報と目標PFCの設定 | S-02 |
